# TonalWave

# How to Run
## Requirements
- [Arduino IDE](https://www.arduino.cc/en/software)
- [Pure Data](https://puredata.info/downloads)

### Note: You might need to configure the ports on the Arduino IDE and Pure Data

1. Open Arduino
   1. Load TonalWave.ino file
   2. Vertify
   3. Upload
2. Open Pure Data
   1. Load A1.pd file
   2. Click on "open 4"
